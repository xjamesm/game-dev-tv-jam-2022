FONTLOG for the Sputnik fonts

This file provides detailed information on the Sputnik Font Software.
This information should be distributed along with the Sputnik fonts and any derivative works.

Basic Font Information

Sputnik is a Unicode typeface family that supports most languages that use the Latin script and its variants, and could be expanded to support other scripts.
